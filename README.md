### About

This is the public-facing marketing website for the RideReady iOS app. It's a static site hosted on GitHub Pages — no frameworks, no build step.

### Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page with features, screenshots, and download CTA |
| `privacy.html` | Privacy policy — RideReady collects zero user data |

### Tech

- Pure HTML, CSS, and vanilla JavaScript
- [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) typefaces
- Scroll-reveal animations via Intersection Observer
- Responsive across mobile, tablet, and desktop
- `.nojekyll` for direct GitHub Pages deployment

### Local Development

Open `index.html` in a browser. That's it.
